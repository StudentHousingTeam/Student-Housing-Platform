## 📌 The Problem

Students struggle to find affordable accommodation. Landlords struggle to find reliable tenants.  
**As'gidle** is a one-stop platform to connect them – built as a **database-driven** learning project.

## 🎯 Project Goals

- ✅ Bring landlords & students together in one place  
- ✅ Offer affordable accommodation listings  
- ✅ Learn to **store**, **retrieve**, and **manipulate** data efficiently  
- ✅ Practice full-stack development with C#/.NET + React  

## 🛠️ Tech Stack (Current)

| Layer | Technology |
|-------|-------------|
| Frontend | React + React Bootstrap |
| Backend | C# / .NET Framework |
